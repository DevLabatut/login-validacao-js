# Login Validação JS

Projeto simples de tela de login feito com **HTML**, **CSS** e **JavaScript**.
O sistema valida e-mail e senha e, se estiver correto, leva o usuário para a página inicial.

## Funcionalidades

* Tela de login
* Validação de campos vazios
* Verificação de e-mail e senha
* Mensagem de erro quando os dados estão incorretos
* Redirecionamento para a página Home após login correto

## Tecnologias usadas

* HTML
* CSS
* JavaScript

## Estrutura do projeto

```
Login-validation/
```

## Como usar

1. Clone o repositório:

```
git clone https://github.com/DevLabatut/login-validacao-js.git
```

2. Abra o arquivo `index.html` no navegador.

3. Use os dados de teste:

```
Email: teste@gmail.com
Senha: teste2026
```

4. Após login correto, você será levado para a página Home.
